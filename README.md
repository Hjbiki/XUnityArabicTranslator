# XUnity Arabic Translator

**مترجم عربي لـ XUnity.AutoTranslator — يترجم ألعاب Unity للعربية مع تشكيل الحروف وترتيب الأسطر تلقائياً.**

العربية | [English](#english)

---

## المشكلة

محركات ألعاب Unity ما تدعم العربية بشكل صحيح:
- الحروف تظهر مفصولة (م ر ح ب ا) بدل متصلة (مرحبا)
- النص يُقرأ من تحت لفوق عند التفاف السطر
- الأقواس ما تنعكس

## الحل

Plugin بصيغة DLL يتحمّل مباشرة داخل XUnity.AutoTranslator — يترجم ويعالج النص العربي تلقائياً:

1. **Arabic Reshaping** — تحويل الحروف لأشكالها الصحيحة (أول، وسط، آخر، منفصل)
2. **BiDi Reordering** — ترتيب النص من اليمين لليسار
3. **Line Reordering** — ترتيب الأسطر من فوق لتحت عبر Harmony Hook يقرأ عرض الـ textbox الفعلي
4. **Lam-Alef Ligatures** — دمج لام + ألف (لا)

## التثبيت

### 1. ثبّت XUnity.AutoTranslator
حمّل من [الإصدارات](https://github.com/bbepis/XUnity.AutoTranslator/releases) — يدعم ReiPatcher أو BepInEx أو MelonLoader.

### 2. حط الـ DLL
انسخ `ArabicGoogleTranslate.dll` إلى:
```
{GameDirectory}/{GameExeName}_Data/Managed/Translators/ArabicGoogleTranslate.dll
{GameDirectory}/{GameExeName}_Data/Managed/ArabicGoogleTranslate.dll
```

### 3. عدّل الإعدادات
في ملف config حق XUnity (يتسوى تلقائي أول تشغيل):
```ini
[Service]
Endpoint=ArabicGoogleTranslate

[General]
Language=ar
FromLanguage=en

[Behaviour]
MaxCharactersPerTranslation=500
MinDialogueChars=2
```

## الإعدادات المتقدمة

```ini
[ArabicGoogle]
ServiceUrl=https://translate.googleapis.com/translate_a/single?client=gtx&sl={0}&tl={1}&dt=t&q={2}
StripTashkeel=True          # إزالة التشكيل (فتحة، ضمة، كسرة)
AutoLineWidth=True          # تقدير عرض السطر تلقائياً (fallback)
LineWidthScale=0.9          # معامل عرض السطر للـ fallback
FixedLineWidth=0            # عرض سطر ثابت (0 = معطّل)
```

## كيف يشتغل

```
النص الأصلي: "The quick brown fox jumps over the lazy dog"
        ↓ Google Translate API
النص المترجم: "الثعلب البني السريع يقفز فوق الكلب الكسول"
        ↓ Arabic Reshaping (presentation forms)
        ↓ BiDi Reversal (RTL → visual LTR)
        ↓ Harmony Hook يقرأ preferredWidth و rectWidth من الـ text component
        ↓ Line wrapping + reordering بناءً على عرض الـ textbox الفعلي
النتيجة: نص عربي مقروء بترتيب صحيح ✓
```

# English

**Arabic translator plugin for XUnity.AutoTranslator — translates Unity games to Arabic with proper character shaping and line ordering.**

## The Problem

Unity game engines don't properly support Arabic:
- Letters appear disconnected instead of joined
- Text reads bottom-to-top when lines wrap
- Brackets don't mirror

## The Solution

A DLL plugin that loads directly into XUnity.AutoTranslator:

1. **Arabic Reshaping** — converts characters to correct positional forms (initial, medial, final, isolated)
2. **BiDi Reordering** — orders text right-to-left for LTR engines
3. **Line Reordering** — fixes line order using a Harmony hook that reads the actual textbox width
4. **Lam-Alef Ligatures** — combines Lam + Alef into proper ligatures

## Installation

1. Install [XUnity.AutoTranslator](https://github.com/bbepis/XUnity.AutoTranslator/releases)
2. Copy `ArabicGoogleTranslate.dll` to both `Managed/Translators/` and `Managed/`
3. Set config: `Endpoint=ArabicGoogleTranslate` and `Language=ar`

## Building

```bash
cd plugin/ArabicTranslatorPlugin
dotnet build -c Release
```
