# XUnity Arabic Translator

**يخلي ألعاب الكمبيوتر تتكلم عربي!**

العربية | [English](#english)

## ايش هالشي؟

Plugin يترجم ألعاب Unity من أي لغة للعربية تلقائياً. تشغّل اللعبة وتلقى الكلام عربي.

> **تنبيه**: الترجمة من Google Translate — يعني أحياناً تكون ركيكة أو غريبة. بس أفضل من ولا شي، خصوصاً لو ما تفهم اللغة الأصلية.

## التثبيت (3 خطوات)

### 1. حمّل XUnity.AutoTranslator
حمّل [أحدث إصدار من هنا](https://github.com/bbepis/XUnity.AutoTranslator/releases) — اختار الملف حسب نوع اللعبة:
- **ReiPatcher** — يشتغل مع أغلب الألعاب (الأسهل)
- **BepInEx** — لو اللعبة تستخدم BepInEx

فك الضغط بمجلد اللعبة وشغّلها مرة.

### 2. حمّل Arabic Translator
حمّل `ArabicGoogleTranslate.dll` من [صفحة الإصدارات](https://github.com/Hjbiki/XUnityArabicTranslator/releases).

حط الملف بمكانين داخل مجلد اللعبة:
```
{اللعبة}_Data/Managed/ArabicGoogleTranslate.dll
{اللعبة}_Data/Managed/Translators/ArabicGoogleTranslate.dll
```

### 3. عدّل الإعدادات
افتح ملف الإعدادات (يتسوّى تلقائي بعد أول تشغيل، عادة بمجلد `AutoTranslator`) وعدّل:
```ini
[Service]
Endpoint=ArabicGoogleTranslate

[General]
Language=ar
FromLanguage=en
```
غيّر `FromLanguage` حسب لغة اللعبة: `en` إنجليزي، `ja` ياباني، `zh` صيني، `ko` كوري.

**شغّل اللعبة واستمتع!**

## أسئلة شائعة

**الحروف مفصولة / الكلام مقلوب؟**
تأكد إنك حاط الـ DLL بالمكانين (Managed + Translators).

**الكلام ما يتغير بعد التحديث؟**
احذف ملف `_AutoGeneratedTranslations.txt` من مجلد Translation.

**يشتغل مع أي لعبة؟**
يشتغل مع ألعاب Unity الي تدعم XUnity.AutoTranslator.

# English

**Automatically translate Unity games to Arabic!**

> **Note**: Translation quality is from Google Translate — it can be rough sometimes, but it's better than nothing when you don't understand the original language.

## Installation

1. Install [XUnity.AutoTranslator](https://github.com/bbepis/XUnity.AutoTranslator/releases)
2. Download `ArabicGoogleTranslate.dll` from [Releases](https://github.com/Hjbiki/XUnityArabicTranslator/releases)
3. Place the DLL in both `Managed/` and `Managed/Translators/`
4. Set config: `Endpoint=ArabicGoogleTranslate` and `Language=ar`

## What it does

- **Arabic Reshaping** — fixes disconnected letters (م ر ح ب ا → مرحبا)
- **RTL Line Ordering** — reads top-to-bottom correctly using actual textbox width detection
- **Bracket Mirroring** — `()` becomes `)(` for Arabic
- **Tashkeel Stripping** — removes diacritics for cleaner display
